============
Installation
============

Install
-------

Download and install:

.. code-block:: bash

   git clone https://github.com/NIB-SI/BoolDoG.git
   cd BoolDoG
   pip install .


Install directly:

.. code=block::bash

   pip insall git+https://github.com/NIB-SI/sBoolDoG.git


Uninstall
---------

.. code-block:: bash

   pip uninstall booldog


Dependencies
------------

The following dependencies are required:

* numpy
* xmltodict
* scipy
* python-igraph
* matplotlib
* pygraphviz (optional)
* PyBoolNet
* networkx
